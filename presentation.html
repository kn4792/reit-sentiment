Project Overview

Research Question

Does employee sentiment predict REIT performance, and did this relationship change after ChatGPT's launch?

Goals

  • Analyze 28,486 Glassdoor reviews from 91 REITs
  • Apply FinBERT sentiment analysis (financial NLP)
  • Test AI adoption effects using ChatGPT launch as natural experiment
  • Examine heterogeneity across property types

Key Innovation

  • First FinBERT application to REIT employee sentiment
  • Largest REIT sentiment dataset created
  • Quasi-experimental design for causal inference
  • Bridges NLP, finance, and organizational behavior

Layer 3: FinBERT Sentiment Analysis

Transformer-Based Classification

Model Architecture

  • Base: BERT (Bidirectional Encoder)
  • Pre-training: 4.9B financial tokens
  • Sources: 10-Ks, earnings calls, analyst reports
  • Fine-tuning: 10,000 labeled financial sentences
  • Layers: 12 transformer blocks
  • Hidden size: 768 dimensions

Implementation Details

  • Framework: PyTorch + Transformers
  • GPU: NVIDIA GTX 1660 Ti (6GB VRAM)
  • Batch size: 32 (optimized for GPU)
  • Sequence length: 512 tokens max
  • Processing time: ~7 minutes total
  • Coverage: 100% classification

Output Schema

  • positive_prob: P(positive sentiment)
  • negative_prob: P(negative sentiment)
  • neutral_prob: P(neutral sentiment)
  • sentiment_label: Categorical {pos, neg, neu}
  • sentiment_score: Continuous [-1, +1]
  • confidence: max(probabilities)

Preliminary Results

Key Findings

  • No average sentiment change post-ChatGPT (p=0.553)
  • Dispersion increased 10.9% (p=0.031**)
  • Review volume surged 89.7% (p<0.001***)
  • Suggests heterogeneous employee responses

Interpretation

Data Center REITs show less negative sentiment (-0.184) vs traditional property types (-0.232 to -0.275). Combined with increased dispersion post-ChatGPT, this supports our heterogeneous treatment hypothesis.

Methodological Contributions

MNIR Failure ‚Üí Valuable Lesson

Initially planned Multinomial Inverse Regression (Campbell & Shang 2022) to measure AI-related language changes

Why MNIR Failed

  • Insufficient time: Only 2 years post-ChatGPT (need ~5)
  • Data imbalance: 66% pre / 34% post (need balanced)
  • Sparse signals: Only 0.26% reviews mention AI
  • Concentration: 59% of AI mentions from 5 Data Center REITs

Why FinBERT Succeeded

  • Semantic capture: 768-dim embeddings vs keyword matching
  • Contextual nuance: Handles opposing interpretations
  • Immediate results: No waiting for language evolution
  • Generalizability: Pre-trained on millions of financial texts

Contribution to Literature

Documenting this negative result provides guidance for researchers studying technology shocks and language evolution with sparse signals and short time horizons
